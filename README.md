# 오사카 회화 PWA

오사카 3박4일 여행용 일본어 회화 학습 PWA.

## 개발

```bash
npm run dev
```

## 배포 (사용자 수동 단계)

1. `npm run build`로 dist/ 생성
2. GitHub 레포 push
3. Vercel 연결 (framework: Vite 자동 감지)
4. iPhone Safari에서 배포 URL 접속 → 공유 → 홈 화면에 추가
